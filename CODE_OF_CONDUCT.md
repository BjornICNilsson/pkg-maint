# Code of Conduct

## Expected Behavior

Please keep interactions around this project:

- respectful
- direct
- constructive
- focused on the work

## Unacceptable Behavior

The following is not acceptable:

- harassment
- personal attacks
- discrimination
- spam
- deliberately disruptive behavior

## Scope

This applies to project discussions, issues, pull requests, and other public interactions related to this repository.

## Enforcement

Project maintainers may remove, edit, or reject contributions and discussions that do not follow this code of conduct.
